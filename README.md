# 🛒 Análisis de Datos de Ventas para Tiendas

## 📌 Propósito del Análisis

El propósito de este proyecto es analizar los datos de ventas generados por diferentes tiendas para:

- Identificar tendencias de compra.
- Detectar productos con mejor y peor desempeño.
- Evaluar el comportamiento de las ventas por tienda, categoría y período.
- Obtener insights que ayuden a optimizar la estrategia comercial y tomar decisiones basadas en datos.

Este análisis busca apoyar a las áreas de marketing, ventas y operaciones en la toma de decisiones estratégicas y tácticas.

## 📉 Resultados

- **Tienda 1** y **Tienda 2** mostraron una tendencia ascendente moderada en sus ventas, con proyecciones que indican estabilidad y potencial de crecimiento.
- **Tienda 3** evidenció un comportamiento mixto, con algunos meses de baja facturación, pero con una proyección positiva a futuro.
- **Tienda 4**, por el contrario, presentó:
  - La **menor facturación acumulada histórica**.
  - Una **tendencia descendente o plana** en sus ingresos.
  - La **proyección de ventas más baja** entre las cuatro tiendas para los próximos 12 meses.

Los resultados muestran de forma clara que Tienda 4 no ha tenido un desempeño competitivo ni se espera que mejore significativamente en el corto plazo.

## ✅ Conclusiones

Tras el análisis comparativo de ventas históricas y la proyección de ingresos futuros, se concluye lo siguiente:

- **La Tienda 4 representa el punto más débil del portafolio**, tanto por su bajo rendimiento actual como por su bajo potencial de crecimiento.
- **Cerrar la Tienda 4 es una decisión financieramente justificada**, ya que permitiría:
  - Optimizar recursos operativos.
  - Reasignar personal, inventario o presupuesto hacia las tiendas con mejor desempeño.
  - Aumentar el enfoque estratégico en las tiendas con mayor retorno esperado.

Este análisis cuantitativo proporciona una base sólida para tomar decisiones de negocio sostenibles y basadas en datos.

## 📌 Herramientas Utilizadas

- Python
  - Pandas
  - NumPy
  - Matplotlib
  - scikit-learn (Regresión Lineal)
- Jupyter Notebook para el desarrollo del análisis
- Visualizaciones gráficas para la interpretación de resultados
