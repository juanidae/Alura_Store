# 🛒 Análisis de Datos de Ventas para Tiendas

## 📌 Propósito del Análisis

El propósito de este proyecto es analizar los datos de ventas generados por diferentes tiendas para:

- Identificar tendencias de compra.
- Detectar productos con mejor y peor desempeño.
- Evaluar el comportamiento de las ventas por tienda, categoría y período.
- Obtener insights que ayuden a optimizar la estrategia comercial y tomar decisiones basadas en datos.

Este análisis busca apoyar a las áreas de marketing, ventas y operaciones en la toma de decisiones estratégicas y tácticas.
